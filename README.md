## 制作アプリのタイトル
クエストToDo

## 制作アプリの説明（40文字程度）
タスク管理をゲーム化。日々のタスク達成が経験値（XP）となり、レベルアップを目指すモチベーション向上アプリ。

## 工夫した点・こだわった点
- ランダムテーマカラーによる新鮮さ
ページをリロードするたびに、Saveボタンや達成マークのアクセントカラーがランダムに切り替わるように工夫。

- RPG風ステータスと達成感
単なるToDoリストではなく、達成したタスクが経験値（XP）として可視化され、レベルアップにつなげる。これにより、タスク完了が単なる作業終了ではなく、「ゲーム内での成長」という報酬へ。

- UXを意識した操作フィードバック
3D風のボタンデザインや、タスク完了時の打ち消し線・薄くなるエフェクト、編集時のスムーズなテキストエリアへの復元など、ユーザーが操作する際の楽しさや快適さを探求してみた。

## 次回トライしたこと（または機能）
- 経験値バー (XP Bar) の視覚化: 現在は「XP / 次レベルまでの必要XP」というテキスト表示ですが、HTMLの<progress>要素やCSSでゲーム風のゲージを実装し、視覚的な達成感をさらに高めたい。

- レベルアップ時の特殊演出: レベルアップ時にアニメーションや効果音を再生し、レベルアップした体験をより印象的なものにしたい。

- 称号システム: 特定のタスク数達成や、特定のレベル到達時に「初心者冒険者」「ベテラン勇者」などの称号を付与する機能。
